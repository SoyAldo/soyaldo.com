nChat

Una excelente opción para establecer el formato del chat de toda tu network. Permite crear canales para el personal, canales de ayuda, etc.

Comandos y Permisos

Comando Permiso Descripción
Spigot
/nchat help nchat.admin Muestra la lista de comandos disponibles.
/nrtpadmin reload nchat.admin Actualiza la configuración y los mensajes.
/nrtpadmin verify nchat.admin Verificar licencia.
/nrtpadmin version nchat.admin Muestra la versión actual del complemento.
/ncommandspy nchat.commandspy Viajar aleatoriamente.
Bungeecord
/nchatbungee help nchat.admin Muestra la lista de comandos disponibles.
/nchatbungee reload nchat.admin Actualiza la configuración y los mensajes.
/nchatbungee verify nchat.admin Verificar licencia.
/nchatbungee version nchat.admin Muestra la versión actual del complemento.
/nprivatemessage [player] [message] nchat.privatemessage Enviar mensaje privado a un jugador.
/nreply nchat.reply Responder al último mensaje privado.
/nprivatemessagespy nchat.privatemessagespy Activar/Desactivar el modo espía de los mensajes privados.
