nRtp

¿Necesitas que tus jugadores viajen por el mundo de manera aleatoria? Pero tienes varios servidores para una sola modalidad, pues nRtp es la solución.

Comandos y Permisos

Comando Permiso Descripción
/nrtpadmin help nrtp.admin Muestra la lista de comandos disponibles.
/nrtpadmin create [type] nrtp.admin Crea un nuevo tipo de viaje aleatorio.
/nrtpadmin delete [type] nrtp.admin Elimina un tipo de viaje aleatorio.
/nrtpadmin list nrtp.admin Enlista los tipos de viajes aleatorios.
/nrtpadmin info [type] nrtp.admin Muestra la información de un tipo específico.
/nrtpadmin edit [type] center nrtp.admin Edita el centro.
/nrtpadmin edit [type] minradius nrtp.admin Edita el radio mínimo.
/nrtpadmin edit [type] maxradius nrtp.admin Edita el radio máximo.
/nrtpadmin reload nrtp.admin Actualiza la configuración y los mensajes.
/nrtpadmin version nrtp.admin Muestra la versión actual del complemento.
/nrtp [type] nrtp.rtp Viajar aleatoriamente.
/nrtp [type] {player} nrtp.rtp.others Enviar aleatoriamente a un jugador.
